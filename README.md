# Agentic AI Security PR Mining

This repository contains code and outputs for mining a dataset of pull requests
generated by agentic AI tools and identifying security-related tasks.

## Data

Place the original dataset CSV files in the `data/` directory:

- `data/all_pull_request.csv`
- `data/all_repository.csv`
- `data/pr_task_type.csv`
- `data/pr_commit_details.csv`

## Tasks

### Task-1

Script: `src/task1.py`  
Input: `data/all_pull_request.csv`  
Output: `outputs/task1_output.csv`  

Columns:
- TITLE
- ID
- AGENTNAME
- BODYSTRING
- REPOID
- REPOURL

### Task-2

Script: `src/task2.py`  
Input: `data/all_repository.csv`  
Output: `outputs/task2_output.csv`  

Columns:
- REPOID
- LANG
- STARS
- REPOURL

### Task-3

Script: `src/task3.py`  
Input: `data/pr_task_type.csv`  
Output: `outputs/task3_output.csv`  

Columns:
- PRID
- PRTITLE
- PRREASON
- PRTYPE
- CONFIDENCE

### Task-4

Script: `src/task4.py`  
Input: `data/pr_commit_details.csv`  
Output: `outputs/task4_output.csv`  

Columns:
- PRID
- PRSHA
- PRCOMMITMESSAGE
- PRFILE
- PRSTATUS
- PRADDS
- PRDELSS
- PRCHANGECOUNT
- PRDIFF

### Task-5

Script: `src/task5.py`  
Inputs:
- `outputs/task1_output.csv`
- `outputs/task3_output.csv`

Output: `outputs/task5_output.csv`  

Columns:
- ID
- AGENT
- TYPE
- CONFIDENCE
- SECURITY (1 = security-related, 0 = not security-related)
